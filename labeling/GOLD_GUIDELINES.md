# Гайдлайн разметки GOLD-сета (NER для поиска М.Видео)

Эталонная разметка, по которой честно меряется качество (словарь и модель). Размечаем **то, что
реально написано в тексте запроса** — сущности, которые пользователь упомянул. Схема — BIO,
три типа: `CATEGORY`, `BRAND`, `ATTR`. Токенизация уже сделана (lowercase, `[0-9a-zа-яё]+`) —
границы менять нельзя, меняем только теги токенов.

Черновик (`gold_review.jsonl`) заполнен нашими словарными правилами — **не доверять ему слепо**,
цель ревью — найти его ошибки. Поправить `tags`, проставить `meta.reviewed=true`, сохранить как
`gold.jsonl`.

## Что размечаем

- **CATEGORY** — слово(а), обозначающие тип товара из топ-10: смартфон/телефон, стиральная
  машина/стиралка, холодильник, iphone/айфон, телевизор, ноутбук/ноут, наушники, пылесос,
  аэрогриль, планшет. Многословную категорию тегируем целиком (`стиральная`=B, `машина`=I).
- **BRAND** — производитель: samsung, xiaomi, haier, bosch, занусси, tcl… Многословный бренд
  целиком.
- **ATTR** — значение характеристики: цвет (`белый`, `черные`), память (`256`, `128 гб`),
  диагональ (`55`, `65 дюймов`), габарит (`60 см`), фича (`5g`, `wifi`, `oled`, `инверторная`),
  тип (`двухкамерный`, `узкая`, `беспроводные`), ОС/CPU (`windows`, `core i5`). Тег в BIO —
  обобщённый `ATTR`; конкретный тип (memory/color/…) при желании писать в `meta.notes` для справки.

## Ключевые правила (фиксируют развилки, всплывшие в черновике)

1. **iPhone / айфон = CATEGORY, не BRAND.** iPhone — это отдельная топ-10 категория. Слова
   `iphone`, `айфон`, `айпад` тегируем `CATEGORY`. `Apple` как таковой (если написан) — `BRAND`.
   (Черновик часто ставит `айфон`→BRAND — это ошибка, править на CATEGORY.)
2. **Тегируем surface, а не «истинную сущность».** Если категория/бренд лишь ПОДРАЗУМЕВАЕТСЯ, но
   слова нет в тексте — НЕ тегируем. Примеры: `wh 1000xm3` (это наушники Sony, но ни «наушники»,
   ни «sony» в тексте нет → всё `O`); `redmi note 14s` (нет слова «смартфон» → категория `O`, но
   `redmi`… см. п.4).
3. **Опечатки и морфоформы тегируем по написанному.** `машинеа`→CATEGORY, `айфор`→CATEGORY(iPhone),
   `черные`→ATTR, даже если словарь их не поймал. Мы размечаем то, что видно.
4. **Модель-коды и серии — `O`.** `s25`, `wh1000xm3`, `43f720t`, `400514039` — идентификаторы, не
   сущности. Слитый с буквами номер — часть модели (`iphone 15` → `15` это поколение → `O`).
   Суб-бренды/линейки (`redmi`, `galaxy`, `airpods`) — на усмотрение: если это узнаваемый бренд в
   поиске (redmi), можно `BRAND`; если это модельная линейка внутри бренда (galaxy) — обычно `O`.
   Решение принять один раз и держать единообразно (рекомендация: redmi/poco/honor = BRAND;
   galaxy/nova/mate/airpods = O).
5. **Числа — ATTR только если это значение характеристики.** `256` в `iphone 256` → ATTR(memory);
   `55` в `телевизор 55` → ATTR(diagonal); но `15` в `iphone 15` → `O` (поколение). Голое число
   без явной привязки к характеристике — `O`.
6. **Границы.** ATTR с единицей — вместе с единицей (`65`+`дюймов` оба I/B-ATTR). Составные фичи
   (`с`, `сушкой`) — по смыслу; предлог сам по себе не сущность (лучше тегировать значимое слово).
7. **Мультибренд/мультикатегория** — размечаем все вхождения (в `занусси … занусси` оба →BRAND).

## Процедура

1. Открыть `gold_review.html` для быстрого визуального просмотра; править — в `gold_review.jsonl`.
2. Для каждого запроса проверить границы и типы, исправить `tags`.
3. Проставить `meta.reviewed=true`. Спорные — пометить в `meta.notes`.
4. Желательно: 30–50 запросов разметить вторым человеком «вслепую» и сверить (inter-annotator
   agreement) — это оценка качества самого gold.
5. Сохранить как `labeling/output/gold.jsonl` (тот же формат `{query_text, tokens, tags, meta}`).
6. Оценка: `python labeling/08_eval_gold.py --gold labeling/output/gold.jsonl` — считает F1
   словарного baseline; позже `--pred model_pred.jsonl` — качество модели, с разрезом
   «словарь поймал / промахнулся».

## Мини-примеры (эталон)

```
стиральная/B-CATEGORY машинка/I-CATEGORY индезит/B-BRAND 6кг/B-ATTR
холодильники/B-CATEGORY черные/B-ATTR
занусси/B-BRAND стиральная/B-CATEGORY машина/I-CATEGORY
купить/O телефон/B-CATEGORY айфон/B-CATEGORY            (айфон -> CATEGORY, не BRAND)
wh/O 1000xm3/O                                          (нет слов сущностей в тексте)
телевизор/B-CATEGORY 55/B-ATTR 4к/B-ATTR samsung/B-BRAND
iphone/B-CATEGORY 15/O pro/O 256/B-ATTR                 (15=поколение -> O; 256=память -> ATTR)
```
